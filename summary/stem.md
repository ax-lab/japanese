# Stem System

The て stem of verbs:

- うつる　→　っ　＋　て、た、たら、たり
- ぬぶむ　→　ん
- くぐ　　→　い
- す　　　→　し

Adjective い stem system:

- く　　→　て、ない、adverb
- かっ　→　た、たら、たり
- けれ　→　ば
- 〇　　→　そう、がる（appear）、すぎる

The copula mini-stem system:
- だっ　→　た、たら、たり
- だろ　→　う
- で、な（kana shifts）

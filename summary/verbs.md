# Verbs

動詞（どうし）

See also: [transitivity](transitivity)

## Groups

- Group 1：五段動詞
- Group 2：一段動詞 (iru/eru)
- 不規則動詞　（変格活用）
	- する
	- くる

## Conjugations

Summary:
- [て](・て):
	- Connective / casual command
- [た](・た):
	- Completed action (e.g. past)
- [あ](stem-あ.md)・[ない](ない)
	- Negative
- え・[る／られる](・える)
	- can do / able to do  (potential)

Forms: 
- 終止形（plain）
- [未然形（ない）](stem-あ)
- [連用形（conjunctive）](stem-masu)
- [連体形（attributive）](stem-attributive)
- 仮定形（hypothetical）
- 命令形（imperative）

### Examples

- 食べる：
	- 食べた　　　→　ate
	- 食べない　　→　not eat
	- 食べられる　→　can eat / able to eat

- ある
	- ない　　　→　don't exist
	- あり得る　→　can exist
- する
	- した　　　→　did
	- しない　　→　not do
	- 出来る　　→　can do / able to do
- 来る
	- きた　　　→　came
	- こない　　→　not come
	- こられる　→　can come / able to come

- 話す：

- 買う：
	- 買った　　→　bought
	- 買わない　→　not buy
	- 買える　　→　can buy / able to buy
- 待つ：
	- 待った　　→　waited
	- 待たない　→　not wait
	- 待てる　　→　can wait / able to wait
- 寝る：
	- 寝った　　→　slept

- 聞く：
	- 聞いて　　→　heard

- 泳ぐ：
	- 泳いで　　→　swam

- 死ぬ：

- 遊ぶ：
